第一次围观网易关于Ui设计师发展的直播

设计师该如何成长？

成长缓慢：

  • 惰性
  • 无指导
  • 路径不正确
  • 团队氛围不佳
  • 个人心态不好
  • 品味和审美不够
  • 过于理性
  • 没有想法
  • 只会抄袭临摹
  • 不愿意沟通
  • 缺乏独立思考能力
  • 不会合理安排时间

结合自身目前而言，自我感觉经过一个月时间，仍然停留在原地的状态。以上的这些点，或多或少都有涉及，直接戳心！目前，最缺乏的应该是实操经历。

目前的状态：

  • 技术不娴熟（一系列软件的操作），自我计划是以PS为主，层层叠加，在有限的时间掌握软件这些工具。
  • 讲不出美与丑，对于别人的设计产品，只能用感知来说话，无法清晰表述作品好在哪里或是哪里不好。
  • 没有规范化和系统化，本身就是野路子出身的学习者，行业信息了解不够广、细。
  • 产品思维尚未形成，目前着手一个花店的WEB端设计，当然，这只是自己的一个练手题目，灵感来自于毕业设计的课题要求。接触到这个题目时，头脑空白，不知道从何入手。于是浏览了一部分国内外的网上花店，在对花店有了一些了解后才开始构想设计风格等。我非常欣赏那些从接触到产品需求后即可形成设计思维的设计师，希望自己能够感知到设计风格的潮流趋势以及设计的精髓，形成自身的一套设计观念。
  • 需要他人的指导，自学者尚未找到专业指导者，仍在自我探索中。这条路好像没有方向，看起来非常漫长，有时候会怀疑自己是否能够坚持。

进阶阶段：

  • 研究美感和设计感

  • 格调和氛围

  • 研究交互动效

  • 关注上下游并思考问题

  • 画面的形式感

  • 色彩搭配构成
