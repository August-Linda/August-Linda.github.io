Events APP 绘制思考总结

发表于 2017-07-26

上次临摹用ps临摹了来自Prague的设计师Jakub Antalik所设计的Events app四张详情页面，奈何迟迟未作总结，今天稍作临摹分析整合。

  • 本次临摹建议画布大小：750*1334px；

  • 重点在于：图标像素对齐，以及目标之间的大小关系；

以下先放出成品图：

Events App

Calendar

User

events app 的页面中正文标题的忽略了要使用环境色；字体的选用的稍稍粗了一点少了一些精致感；calendar 和user的分割线给的稍粗了一些，另外还是给的纯灰没有加上环境色；右上角3这里的投影过重，另外从图片上来看底部的青色板块好像没有居底对齐；user界面中小女孩的眼睛正面对着会有些干扰内容的阅读，另外底部青色版本这里的投影不小心漏掉了。

这次的临摹又出现了和上一次一样的问题，那就是icon的精细化处理，以及字体的选择，做着设计的工作，应该严格要求自己，使界面的和实用性和美观性都由较好的表达。两次临摹界面均出现了这些缺点，这其实更说明目前我的配色，字体运用能力还不达标，还要继续探索。

这一次出现的环境色运用问题足以让我重视场景的融合，不可以主观臆造界面，而应该依据实际页面信息。在临摹时，觉得遮罩部分始终做得不够通透，最后发现这是未考虑环境导致的，在下一次的临摹中将会尝试使用带有一定环境色的遮罩层。
